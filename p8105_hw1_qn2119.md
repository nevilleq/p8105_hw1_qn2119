Problem 1
=========

First, we will create a data frame `variable.types.df`, containing a random sample variable from a random variable *X* ∼ *U**n**i**f**o**r**m*(0, 5), a logical variable describing whether those elements are greater than two, and a character variable from a random sample from \[*Z**e**r**o*, *O**n**e*, *T**w**o*\], and a coerced factor variable of the previous variable.

``` r
#Defining number of columns in data frame
n.col.df <- 10

#Creating our data frame with rUnif(0,5), logical, character random sample, and factor vars
variable.types.df <- tibble(
  unif.sample = runif(n.col.df, min = 0, max = 5),
  logical.unif.g.2 = ifelse(unif.sample > 2, TRUE, FALSE),
  char.sample = sample(c("0", "1", "2"), n.col.df, replace = TRUE),
  factor.char.sample = as.factor(char.sample)
)
```
